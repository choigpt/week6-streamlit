# Streamlit deployment

Create one GitHub repository that contains:

- `q2_app.py`
- `q3_app.py`
- `q5_app.py`
- `requirements.txt`
- `data/sunspots.csv`

Then create three Streamlit Cloud apps from the same repository:

1. App for Q2
   - Main file path: `q2_app.py`
   - App URL/name: your name + `2`

2. App for Q3
   - Main file path: `q3_app.py`
   - App URL/name: your name + `3`

3. App for Q5
   - Main file path: `q5_app.py`
   - App URL/name: your name + `5`

After deployment, paste each generated URL into the corresponding notebook cell.
